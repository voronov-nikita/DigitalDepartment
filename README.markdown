## Введение

Цифровая кафедра — уникальная возможность для студентов не-IT направлений подготовки и специальностей получить дополнительное IT-образование параллельно с освоением основной образовательной программы.

Проект «Цифровая кафедра» реализуется в рамках федерального проекта «Университеты для поколения лидеров» национального проекта «Молодёжь и дети», целью которого является обеспечение приоритетных отраслей экономики высококвалифицированными кадрами, обладающими цифровыми компетенциями.

Вам представляется уникальная возможность приобретения новых IT-знаний и навыков, которые позволят стать высоко востребованными специалистами на рынке труда, получить интересную работу и выстроить успешную профессиональную карьеру.

Преимущества обучения на Цифровой кафедре:
Возможность получить и углубить знания в самых востребованных IT-направлениях
Возможность участия в мероприятиях по знакомству с крупнейшими работодателями IT-индустрии по всей России
Получение диплома профессиональной переподготовки с присвоением новой IT-квалификации
Получение дополнительных баллов для поступления в магистратуру РТУ МИРЭА
Гибкий график обучения в онлайн и офлайн форматах
Полностью бесплатное обучение
В 2025-2026 учебном году РТУ МИРЭА продолжает реализацию программ дополнительного профессионального IT-образования с целью обеспечения приоритетных отраслей экономики высококвалифицированными кадрами, обладающими цифровыми компетенциями.

В разработке и реализации образовательных программ принимают участие ведущие специалисты из крупных российских IT-компаний. В рамках Цифровой кафедры приглашённые специалисты читают лекции, помогают студентам реализовать собственные проекты и проходить практическую подготовку.

### Создание базового HTML

Язык, который понимают браузеры - это java script, а то, что понимают пользователи - это HTML - язык гипертекстовой разметки. Его обычно не называют языком программирования, так как в нем даже нельзя создать переменную, можно только создавать структуру для отображения какого-то контента на сайте.

Базовый HTML код выглядит в формате:

```html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
<body></body>
</html>

```
